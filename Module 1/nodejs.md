# 📘 Note 1

## 🟢 Node.js

> **Node.js** is a **runtime environment** that allows JavaScript to run outside the browser.

---

### 🔹 Key Points

1. Node.js allows you to run JavaScript **outside the browser**.
2. It enables JavaScript to **interact with the system (OS)** using underlying **C++ bindings**.
3. You can build **web servers** using JavaScript.
4. It extends JavaScript beyond frontend development into **backend development**.

---

### ⚙️ JavaScript Engines

A **JavaScript engine** is a program that executes JavaScript code.

Examples:

1. **Google Chrome** → V8 Engine
2. **Mozilla Firefox** → SpiderMonkey Engine

---

### 🧠 Summary

* Node.js = Runtime for executing JS outside browser
* Uses powerful engines like **V8**
* Enables backend development with JavaScript
* Bridges JavaScript with system-level operations
